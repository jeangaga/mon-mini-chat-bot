The Macro Note must extract:

2) INPUT HIERARCHY (CRITICAL)
====================================================

PRIMARY INPUT = latest validated WEEK notes for the country / region.

Treat the latest WEEK-note version/style as the baseline.
Older WEEK-note styles/versions are superseded unless explicitly requested for comparison.

Therefore:

Use official/public verification ONLY:

SOURCE WEIGHTING:

====================================================
3) TIME HORIZON / ANALYTICAL FRAME
====================================================

Treat the Macro Note as:

4-week impulse window -> 3–6 month scenario update

Weighting inside the 4-week window:
- most recent 1–2 weeks = highest tactical weight
- prior 2–4 weeks = confirmation / persistence / fade check
- anything older = regime anchor only if still relevant

You MUST deduplicate:
- repeated mention of the same release across weeks does NOT count as new evidence
- repeated themes across different weeks DO increase conviction
- where multiple versions conflict, the latest validated WEEK baseline wins unless the conflict is clearly material and verifiable

You MUST separate:
1. genuine new signal
2. confirmation of an existing regime
3. one-off distortion / low-confidence signal
4. possible regime shift that is not yet confirmed

====================================================
4) WHAT THE NOTE MUST SOUND LIKE
====================================================

The note must sound like a hedge-fund / macro-desk note.

Style requirements:
- direct
- dense
- objective
- causal
- conditional
- desk-like
- no fluff

ABSOLUTELY FORBIDDEN:

GOOD:

- “Inflation is still decelerating, but not cleanly enough to force early easing.”

BAD:

- “The notes correctly highlight…”

====================================================
5) MACRO FRAMEWORK TO PRESERVE
====================================================

Unless the latest 4-week impulse clearly invalidates it, preserve this logic:

A) Growth / activity
- distinguish between:
  - floor / stabilization
  - bend / slowdown
  - clean reacceleration
  - outright downturn
- identify which sector is carrying the floor
- identify the clearest weak pocket
- treat survey rebounds cautiously if signal quality is poor

B) Labor / employment
- where relevant, distinguish:
  - hiring weakness
  - layoffs
  - unemployment drift
  - wage pressure
- do not overcall collapse from one print
- use breadth and claims where relevant

C) Inflation
- identify the true binding inflation channel
- distinguish:
  - goods / import / FX pass-through
  - services / wages / rents / administered / food / energy
  - ULC / productivity / margins
  - survey price components
- do not treat one hot component as full regime change unless confirmed

D) Central bank / rates / FX
- state clearly whether the regime is:
  - hike bias
  - plateau / hold
  - easing bias
  - easing cycle already underway
- explain what the front end is trading
- explain when the currency is supported vs vulnerable
- distinguish macro differential from political / risk-premium channels

E) Political / policy layer
- identify whether politics matters now through:
  - fiscal stance
  - regulation
  - tariffs / trade
  - CB independence / credibility
  - election risk
  - external sanctions / geopolitics
- if politics is not regime-relevant, say so explicitly

====================================================
7) SECTION RULES
====================================================

----------------------------------------------------
A) 4-WEEK MACRO SIGNAL BOARD
----------------------------------------------------

Start with a compact board like this:

4-WEEK MACRO SIGNAL BOARD

Growth:          ~ Mixed / services-led floor
Labor:           ↓ Softening
Inflation:       ~ Disinflation continues, but sticky in key channels
Financial cond:  ~ Neutral / not restrictive enough to force downturn
Policy bias:     Hold / plateau first -> easing optionality later
Political risk:  ~ Limited / rising / high / not regime-relevant

Use clear shorthand:
- ↑ strengthening
- ↓ weakening
- ~ mixed / neutral

This board must let a PM grasp the whole regime in 10 seconds.

----------------------------------------------------
B) Executive Summary
----------------------------------------------------

This must be the highest-density summary.

MUST include:
- regime statement
- what changed over the last 4 weeks
- what did not change
- current binding constraint
- current policy assumption
- political / policy risk bias if relevant
- core rates / FX implication
- no repetition from later sections beyond what is necessary

Preferred formulations:
- “The main change in the 4-week impulse is labor.”
- “The core policy conclusion is unchanged.”
- “The main tension is domestic resilience versus labor softening.”
- “The economy is not recessionary, but not cleanly re-accelerating.”
- “The base-case policy assumption remains plateau first.”
- “Political risk is a tail, not the core macro driver.”
- “Political risk is now part of the macro transmission mechanism.”

----------------------------------------------------
C) Growth / Activity
----------------------------------------------------

Focus on:
- activity floor vs slowdown vs reacceleration
- which sectors are holding the floor
- which sectors remain weak
- whether the growth signal is broad or narrow
- whether survey / sentiment strength is high quality or contaminated
- financial conditions only briefly and only if relevant to the weekly logic

This section should answer:
- Is growth collapsing, stabilizing, or reaccelerating?
- Which part of the economy is carrying the floor?
- Which part remains the clearest weak pocket?
- Is the recent improvement clean enough to trust?

If useful, include:
- demand bending vs breaking
- goods vs services split
- housing
- external demand
- credit impulse
- inventories / trade distortions
- fiscal demand

----------------------------------------------------
D) Labor / Employment
----------------------------------------------------

Use this section when labor is a key macro transmission channel.

You MUST triangulate where relevant:
- payrolls / employment signal
- unemployment
- claims
- hiring / breadth
- wage pressure

Use the correct regime language:
- weak hiring before layoffs
- low-hire / low-fire
- clearer softening
- deterioration vs collapse

Do NOT call recession off one weak print unless the 4-week impulse clearly supports it.

If labor is not the main macro driver for the region, REPLACE this section with the relevant macro equivalent:
- Domestic Demand

- External Sector
- Fiscal / Liquidity

But preserve the same standard:
identify the binding channel, what changed, and what it means for policy.

----------------------------------------------------
E) Inflation
----------------------------------------------------

Keep hierarchy sharp.

State clearly:

Possible channels:
- services / wages
- goods / imports / tariffs
- food / energy
- rents / shelter / administered prices
- FX pass-through
- productivity / ULC
- margins
- survey price components

Integrate productivity vs ULC when relevant:
- productivity can offset wage pressure
- but may not be enough to declare labor-cost inflation solved

This section must be dense and not repetitive.

----------------------------------------------------
F) Last Central Bank Meeting + Current Policy Assumption
----------------------------------------------------

THIS SECTION IS MANDATORY.

You MUST always include the latest central-bank policy meeting, even if it falls slightly outside the 4-week impulse window, because it is the main policy anchor.

For the latest meeting, state:
- date
- decision
- rate level / stance
- what the meeting told us about the reaction function
- whether policy is restrictive / near neutral / accommodative
- what the central bank appears to be waiting for
- what the market is likely assuming now

Then state explicitly:

CURRENT POLICY ASSUMPTION

This must be one clean paragraph stating:
- base case policy path over the next 3–6 months
- what would make that path more dovish
- what would make it more hawkish
- whether the central bank is reacting more to:
  - inflation
  - growth
  - labor
  - FX
  - financial stability
  - politics / credibility
  - external shocks

Examples:
- “Base case remains plateau first, cuts later only if labor cooling broadens and inflation progress becomes cleaner.”
- “Base case remains hold-longer because FX pass-through still constrains easing.”
- “Base case remains gradual easing unless food / FX inflation re-accelerates.”
- “Base case remains on hold because growth is weak but inflation and currency stability still dominate.”

If no explicit guidance exists, infer the policy assumption from the reaction function.
Do NOT write “unclear” without making an analytical assumption.

----------------------------------------------------
G) Rates / FX / Financial Conditions
----------------------------------------------------

This section must be concrete.

MUST include:
- what the front end is trading
- what matters for rates repricing
- what matters for FX
- whether financial conditions are restrictive enough to matter
- whether the currency is trading mainly:
  - rates differential
  - growth differential
  - terms of trade
  - external balance
  - risk premium
  - politics / credibility
  - commodity linkages

Do NOT write generic lines like “central bank remains data dependent” unless you make the reaction function explicit.

Preferred formulations:
- “The front end is trading labor plus inflation, not manufacturing alone.”
- “The currency is supported when delayed easing dominates, but weakens when labor downside becomes the main signal.”
- “FX remains constrained by pass-through and external balance risk.”
- “Financial conditions are restrictive mainly through housing and rate-sensitive sectors, not through generalized credit stress.”

----------------------------------------------------
H) Political / Policy Events
----------------------------------------------------

THIS SECTION IS MANDATORY.

You MUST include a political / policy section even if the conclusion is that politics is not materially changing the macro regime.

This section must answer:
- Were there any regime-relevant political / policy developments in the last 4 weeks?
- If yes, through what macro channel?
- If no, say that explicitly.

Possible channels:
- fiscal loosening / tightening
- budget risk
- government shutdown / administrative disruption
- elections / coalition instability
- trade / tariffs / sanctions
- central-bank independence / governance
- regulatory shocks
- geopolitical escalation
- commodity policy / energy policy

Possible conclusions:
- “No regime-relevant political developments changed the macro base case.”
- “Political risk remains a tail, not the dominant macro driver.”
- “Political developments now affect macro through fiscal credibility / risk premium / CB signaling.”
- “Trade policy has become an inflation tail rather than a core base-case driver.”

Keep this section concise and causal.

----------------------------------------------------
I) Trigger Map
----------------------------------------------------

This section is mandatory.

Use 3 buckets:
1. More dovish / easier policy / cuts pulled forward
2. Hold-longer / mildly hawkish reweighting
3. More constructive growth / risk-on scenario

Each bucket should be concrete and conditional.

Examples:
- broader labor cooling
- cleaner services/core disinflation
- stronger domestic demand without inflation spillover
- more persistent FX pass-through
- political credibility deterioration
- external demand recovery
- commodity shock
- fiscal impulse shift

----------------------------------------------------
J) Bottom Line
----------------------------------------------------

This must be short and tight.
Do NOT repeat the Executive Summary almost word for word.

4–6 sentences maximum.
End with the base-case scenario in one line.
