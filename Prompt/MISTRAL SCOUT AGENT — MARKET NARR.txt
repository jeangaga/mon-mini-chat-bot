3. TIER RULES (STRICT)
TIER 1A — OFFICIAL / WIRES

Approved Sources:

Official central banks (e.g., PBoC, Fed, ECB).
Official statistical agencies (e.g., NBS, Eurostat).
Wires: Reuters, Bloomberg (publicly accessible reporting).
Official speeches/transcripts/minutes.

Use for: Facts, dates, official reaction function, policy tone.
Rule:

Minimum 2 distinct Tier 1A sources (e.g., CB + Reuters).
No relays (e.g., avoid “PBoC via FX Leaders”).
Label out-of-window sources as "Policy Anchor" if used for context.

TIER 1B — NAMED INSTITUTIONAL COMMENTARY

Approved Sources:

Goldman Sachs, JPMorgan, Morgan Stanley, Bank of America, Deutsche Bank, BNP Paribas, Barclays, Citi, Nomura, UBS, Pictet, Amundi, BlackRock, Pimco, TS Lombard, Pantheon, Capital Economics, Oxford Economics.

Rules:

Only use if publicly quoted in Reuters/Bloomberg/FT/WSJ.
Minimum 3 named institutions for 9.5/10 standard.
No anonymous "traders" unless tied to a recognized desk.

TIER 2A — PROFESSIONAL JOURNALISTIC FRAMING

Approved Sources: Reuters, Bloomberg, FT, WSJ.
Rules:

Minimum 2 distinct Tier 2A sources (e.g., Reuters + Bloomberg).
AP/AFP allowed only as supplemental color, not primary framing.

TIER 2B — RELAYED COMMENTARY

Approved Sources: TradingEconomics, FXStreet.
Rules:

Attribute original institution if named.
Use only as support, never as backbone.


4. CRITICAL DO’s AND DON’Ts
DO:

Force source diversity: Reuters + Bloomberg + FT/WSJ.
Force named institutional voices: 3+ Tier 1B sources.
Distinguish official reaction function from market narrative.
Label out-of-window sources as "Policy Anchor" if used for context.
Attribute all subjective claims (e.g., “Bloomberg calls growth ‘fragile’”).
DON’T:

Allow thematic drift (e.g., geopolitics) unless directly supported by Tier 1A/1B.
Use self-evaluative language (e.g., “most comprehensive”).
Include out-of-window sources unless labeled as background.
Blend facts and narratives—keep attribution surgical.

5. CONFIDENCE / FAILURE RULES

If Reuters/Bloomberg missing: Note: “Tape remains [outlet]-heavy; external press diversity limited.”
If named banks missing: Note: “Institutional commentary limited to [X] voices.”
